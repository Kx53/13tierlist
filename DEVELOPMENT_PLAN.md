# 🚀 13TierList - Future Development Plan (Phase 5+)

เอกสารนี้ใช้สำหรับจดบันทึกแผนงานและฟีเจอร์ต่างๆ ที่เตรียมไว้สำหรับการพัฒนาในอนาคต (Major Upgrades)

## 📌 แผนการอัปเกรดแบบยกเครื่อง (Major Overhaul)

### 1. Backend Migration (Elysia.js)
- โละ `Express.js` ทิ้ง และย้ายไปใช้ **Elysia.js** แบบเต็มตัว
- **เหตุผล:** เพื่อดึงประสิทธิภาพสูงสุดของ `Bun` ออกมาใช้ (เร็วกว่า Express หลายเท่าตัว) และยังรองรับ Eden RPC สำหรับ Type-Safety เต็มรูปแบบระหว่างหน้าบ้านกับหลังบ้าน

### 2. Frontend UI Framework (HeroUI v3)
- นำระบบ UI Library เข้ามาใช้งานแทนที่การเขียน Tailwind CSS แบบดิบๆ บางส่วน
- **ตัวเลือกหลัก:** `HeroUI v3` (อ้างอิงและใช้งานคู่กับระบบเอกสารจาก MCP Server)
- **เป้าหมาย:** ทำปุ่ม, Modal, หรือ Component ต่างๆ ให้มีความพรีเมียม สวยงาม อนิเมชันเนียนตา และช่วยลดโค้ดฝั่ง UI ให้น้อยลง

### 3. Professional Iconography (Lucide-React)
- เลิกใช้อีโมจิแบบแข็งๆ (เช่น ✏️, ❌, 🗑️, ➕)
- **ตัวเลือกหลัก:** `lucide-react`
- **เป้าหมาย:** สลับไปใช้ไอคอนที่เป็น Vector (SVG) เพื่อความสวยงาม เป็นมืออาชีพ ควบคุมขนาด-สีผ่าน Tailwind ได้ง่ายกว่า และไม่ต้องลงไลบรารีของ Astro ฝั่งเซิร์ฟเวอร์ซ้ำกระตุ้นให้เครื่องหนัก

### 4. Internationalization (i18n)
- สร้างระบบสองภาษา (รองรับ **TH** และ **EN**)
- **ตัวเลือกหลัก:** `@nanostores/i18n` (สำหรับระบบ State-based เบาๆ) 
- **เป้าหมาย:** ผู้ใช้งานสามารถกดปุ่มสลับภาษาเพื่อเปลี่ยน UI จากไทยเป็นอังกฤษ (หรือสลับกลับ) ได้ทันทีแบบ **Real-time** โดยหน้าเว็บจะไม่กระตุกหรือรีเฟรช และไม่กวนใจเวลาที่กำลังเมามันกับการลากวาง Tier List อยู่
